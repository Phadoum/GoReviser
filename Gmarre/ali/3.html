Nombres complexes





1) Définition-Propriétés

  • Un nombre complexe ƶ s'écrit de façon unique sous la forme a + ⅰ𝑏 ;a ∈ ℝ ; 𝑏 ∈ ℝ.
    On dit que a + ⅰ𝑏 est la forme algébrique du nombre complexe ƶ.
    a est la partie réelle de ƶ, on note a = ℜ𝖊(𝖟)
    𝑏 est la partie imaginaire de ƶ, de ƶ, on note ɓ = ℑ𝘮(𝖟)

  • Le nombre complexe ⅰ est tel que ⅰ² = −1.
    Les complexes de la forme 𝑏ⅰ avec 𝑏 ∈ IR,sont appelés imaginaires purs.

  • Deux nombres complexes sont égaux si et seulement si ils ont même partie réelle et même partie imaginare.

2) Équation du second degré à coefficients réels

  • L'équation aƶ² + 𝑏ƶ + 𝑐 = 0, où a, 𝑏 et a sont des réels (avec a ≠ 0) admet dans ℂ deux solutions (éventuellement confondues).
    Soit ∆ = 𝑏² - 4a𝑐 le discriminant de l'équation. ∆ est un nombre réel.
    - si ∆ ≥ 0, les deux solutionns sont réelles: ƶ1 = −𝑏 − √∆ ⁄2a    et    ƶ2 = −𝑏 + √∆ ⁄2a
    - si ∆ ≤ 0, on peut écrire ∆ = (ⅰδ)² avec δ ∈ ℝ, les deux solutions sont alors des nombres complexes(conjugués l'un de l'autre)
  • ƶ1 = −𝑏 −ⅰδ ⁄2a et ƶ2 = −𝑏+ ⅰδ ⁄2a
    Le trinome aƶ² + 𝑏ƶ + 𝑐 se factorise sous la forme a(ƶ−ƶ𝟙)(ƶ−ƶ𝟚)

3) Module et conjugué d'un nombre complexe

  • On appelle module du nombre complexe ƶ = a + ⅰ𝑏, a ∈ ℝ, 𝑏 ∈ ℝ, le réel positif ⎸ƶ⎹ = √a²+𝑏².
    ⎸ƶ⎹ = 0 ⇔ ƶ= 0    ;    ⎸−ƶ⎹ = ⎸ƶ⎹    ;    ⎸ƶ+ƶ'⎹ ≤ ⎸ƶ⎹ + ⎸ƶ'⎹
    ⎸ƶƶ'⎹ = ⎸ƶ⎹ . ⎸ƶ'⎹ ;   ⎸1⁄ƶ⎹ = 1⁄⎸ƶ⎹   ;    ⎸ƶ⁄ƶ'⎹ = ⎸ƶ⎹ ⁄ ⎸ƶ'⎹
  • On appelle conjugué du nombre complexe ƶ⎺=a - ⅰ𝑏

4) Forme trigonométrique d'un nombre complexe non nul

  • Tout nombre nombre complexe non nul ƶ peu être écrit sous la forme : ƶ=𝘳 (cos θ + ⅰ sin θ) avec θ ∈ ℝ et 𝘳 ∈ ℝ+*.
    C'est la forme trigonométrique de ƶ
    𝘳 est le module de ƶ, 𝘳 = ⎸ƶ⎹
    θ est un argument de ƶ.
    Si ƶ=𝘳 (cos θ + ⅰ sin θ) alors
    ∘ ƶ⎺ = 𝘳 (cos (╺θ) + ⅰ sin (╺θ))
    ∘ ╺ƶ=𝘳 (cos (θ+π)+ⅰ sinθ+π)

5) Argument d'un nombre complexe

  • L'argument d'un nombre complexe ƶ n'est pas unique, il est défini modulo 2π.
    Si θ est un argument de ƶ, on notera 𝔞𝔯𝔤 ƶ= θ [2π] ou 𝔞𝔯𝔤 ƶ= θ + 2𝑘π( 𝑘∈ℤ)
    On appelle argument principal de ƶ l'argument de ƶ appartenant à ]╺π;π].

  • Pour ƶ ∈ ℂ* et ƶ' ∈ ℂ*, on a ƶ=ƶ' ⇔ ⎰ ⎸ƶ⎹ = ⎸ƶ'⎹
    •                                                                 ⎱ 𝔞𝔯𝔤 ƶ = 𝔞𝔯𝔤 ƶ' [2π]
